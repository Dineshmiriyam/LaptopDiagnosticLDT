Smart Diagnosis Engine
9-PHASE STATE MACHINE — OPTION 54
PHASE 0
PREFLIGHT
  • Admin verification (WindowsPrincipal)
  • PowerShell 5.1+ check
  • Session GUID generation
  • System Restore point
  • HKLM\SYSTEM + SOFTWARE export
  • BCD backup + manifest
  • Abort on any failure
PHASE 1
SNAPSHOT
  • Machine identity (serial, vendor, model)
  • OS + build + activation status
  • BIOS + firmware type (UEFI/Legacy)
  • Chassis type (Laptop/Desktop/Tablet)
  • RAM, disks, battery, uptime
  • Pending update count
  • Event log errors (System + App)
  • Full driver + service inventory
  • Output: snapshot.json
PHASE 2
HARDWARE INTEGRITY
  • SMART disk health + wear level
  • WHEA memory errors (5+ = Fail)
  • Thermal sensors (>85C = Critical)
  • Battery wear (>60% = Fail)
  • TPM presence + readiness
  • PnP device errors
  • HardwareCritical gate (blocks 3-7)
PHASE 3
BOOT & OS
  • BSOD analysis (minidumps + BugCheck)
  • Unexpected resets (KP 41 + 6008)
  • SFC integrity (CBS.log scan)
  • DISM CheckHealth
  • Secure Boot status
  • Windows Update age
  • BCD validation (bcdedit /enum)
  • CHKDSK dirty bit (fsutil)
  • Corrupt services (missing binaries)
  • WU component health
PHASE 4
SERVICE, DRIVER & DISPLAY
  • Problem devices (PnP error codes)
  • Unsigned drivers
  • Critical services (Defender, FW, WU)
  • Defender RTP + signature age
  • Firewall profiles (all 3)
  • Network adapters (Wi-Fi/Ethernet)
  • Suspicious scheduled tasks
  • GPU adapter health + driver age
  • GPU TDR/crash events (4101/4116)
  • Display panel + cable indicators
PHASE 5
PERFORMANCE
  • CPU benchmark (10s iteration count)
  • Memory pressure (>90% = Fail)
  • Disk free space (<10% = Fail)
  • Disk I/O benchmark (MB/s)
  • Startup items (>15 = Warning)
  • Default gateway reachability
  • Winsock catalogue health
  • Proxy misconfiguration
  • Stores PerfBaseline for Phase 7
PHASE 6
REMEDIATION
  • 3 gates: HW check, L1 max, 60% conf
  • Pre-remediation backup created
  • Dispatch: BootRepair
  • Dispatch: BSODTroubleshooter
  • Dispatch: DriverRepair
  • Dispatch: SecurityHardening
  • Dispatch: NetworkTroubleshooter
  • Dispatch: SoftwareCleanup
  • Each result: SUCCESS / FAILED
PHASE 7
STRESS VALIDATION
  • Pre-stress thermal baseline
  • CPU stress (configurable duration)
  • Memory allocation + integrity check
  • Disk I/O stress (baseline compare)
  • Post-stress event log check
  • Memory diagnostic recommendation
  • Thermal delta comparison
  • PASS / FAIL verdict
PHASE 8
CLASSIFICATION
  • Primary root cause (highest weight)
  • Secondary symptoms (2-4 items)
  • Enterprise score 0-100
  • Confidence %
  • Escalation level (None/L1/L2/L3)
  • Terminal summary + progress bar
  • Output: final_report.json + HTML

Escalation Model
3-LEVEL ESCALATION TAXONOMY
Level 1 — Auto-Fixable
SOFTWARE
  • Defender RTP re-enable
  • Firewall profiles re-enable (3 profiles)
  • Network stack reset (Winsock + IP)
  • DISM RestoreHealth + SFC repair
  • WU component reset
  • Driver repair dispatch
  • BSOD troubleshooter dispatch
  • Startup cleanup dispatch
  • Security hardening dispatch
Level 2 — Replaceable
COMPONENT
  • Battery wear >60%
  • SMART critical / drive Unhealthy
  • NVMe wear >80%
  • WHEA correctable RAM errors
  • CMOS battery dead
  • Display artifacts (no damage) — eDP cable / panel
Level 3 — Technician
REQUIRED
  • WHEA uncorrectable memory
  • No POST / dead board
  • BIOS corruption
  • TPM absent (required machine)
  • Storage controller failure
